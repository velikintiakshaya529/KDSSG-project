# KDSSG – SQL & Azure Trainings Website

A static, responsive landing page for KDSSG showcasing SQL Server DBA and Azure trainings, institute highlights, learner reviews, and contact details. Built using HTML5 and CSS3.

## 📌 Overview

- Project Type: Static Website (No JavaScript framework)
- Primary File: index.html
- Styles: css/stylesheet2.css
- Assets Folder: images/ (icons, logos, banners, course images)

## 📚 Sections Included

- Top contact strip (Phone & Email)
- Main navigation with logo and links
- Hero banner (Call to “Join Now”)
- About KDSSG – Center for Excellence introduction
- Why KDSSG – Benefits & quick highlights
Trainings –
  - SQL Server DBA + Azure Cloud
  - Azure for Experienced DBAs
- Reviews & video review preview
- Footer with location and social links
